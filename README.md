# 📻 Morse Code Training Pro

เว็บแอปพลิเคชันสำหรับฝึกฟังและฝึกเคาะรหัสมอร์ส (Morse Code) รูปแบบสไตล์ Modern Tactical / Cyberpunk Minimal ออกแบบมาเพื่อให้ผู้ที่สนใจและนักวิทยุสมัครเล่นสามารถพัฒนาทักษะการรับ-ส่งรหัสมอร์สได้ด้วยตนเองผ่านเว็บเบราว์เซอร์

---

## 🌟 คุณสมบัติเด่น (Features)

* **โหมดฝึกฟัง (Listening Mode):** * สุ่มตัวอักษรภาษาอังกฤษ (A-Z) หรือตัวเลข (0-9) ครั้งละ 5 ตัวอักษร
    * ปรับความเร็วในการส่งสัญญาณได้ตั้งแต่ **5 WPM ถึง 30 WPM** (Words Per Minute)
    * ระบบตรวจสอบคำตอบพร้อมเฉลยในกรณีที่ตอบผิด
* **โหมดเคาะอิสระ (Straight Key Simulator):**
    * จำลองการเคาะรหัสจริงผ่านปุ่มคลิกค้างบนหน้าจอ หรือกดปุ่ม `Spacebar` บนคีย์บอร์ด
    * รองรับการใช้งานบนหน้าจอสัมผัส (Touch Screen) บนสมาร์ตโฟนและแท็บเล็ต
    * ระบบเรืองแสงตามจังหวะเคาะจริง (Interactive Feedback)
    * แปลงรหัสสัญญาณ (จุด `.` และ ขีด `-`) ออกมาเป็นตัวอักษรและบันทึกประวัติข้อความยาวต่อเนื่องได้อัตโนมัติ
* **คู่มือรหัสมอร์สอ้างอิง (Reference Guide):**
    * ตารางอ้างอิงรหัสมอร์ส (A-Z, 0-9 และเครื่องหมายพิเศษ) แสดงผลแบบ Dynamic เข้าใจง่ายในหน้าเดียว
* **UI/UX ระดับพรีเมียม:**
    * สร้างด้วยเทคโนโลยี **Tailwind CSS** และใช้หลักการ **Glassmorphism** ผสมผสานเอฟเฟกต์ Glow เพิ่มความน่าใช้งาน
    * ป้องกันปัญหาระบบเคาะทำงานซ้ำซ้อนขณะที่ผู้ใช้งานกำลังพิมพ์คำตอบในช่องข้อความ

---

## 🛠 เทคโนโลยีที่เลือกใช้ (Tech Stack)

* **HTML5 & CSS3** - โครงสร้างและการจัดหน้าเว็บ
* **Tailwind CSS** - สำหรับการออกแบบสไตล์สถาปัตยกรรม UI ที่รวดเร็วและสวยงาม (Responsive Design)
* **Vanilla JavaScript (ES6+)** - ควบคุมตรรกะระบบทั้งหมดของแอปพลิเคชัน
* **Web Audio API (AudioContext)** - ใช้ในการสร้างคลื่นความถี่เสียงและควบคุมจังหวะเสียงสั้น-ยาว (Dot & Dash) โดยไม่ต้องพึ่งพาไฟล์เสียงภายนอก

---

## 🚀 วิธีการใช้งาน (Getting Started)

1. ดาวน์โหลดหรือคัดลอกไฟล์ซอร์สโค้ด `index.html`
2. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ยุคปัจจุบันได้ทันที (เช่น Google Chrome, Microsoft Edge, Safari หรือ Firefox) โดย**ไม่ต้องติดตั้งเซิร์ฟเวอร์หรือลงโปรแกรมเพิ่มเติม**
